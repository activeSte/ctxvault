# Product Documentation

## Getting Started

### Creating an account
Visit app.example.com and click "Sign up". Enter your email and choose a password. You will receive a verification email — click the link to activate your account.

### Dashboard overview
The dashboard shows your usage metrics, recent activity, and quick actions. The left sidebar contains navigation to all main features: Projects, Analytics, Integrations, and Settings.

### Creating your first project
Click "New Project" from the dashboard. Enter a project name and optional description. Choose a template or start from scratch. Projects are private by default.

## Integrations

### Slack
Connect your workspace via Settings > Integrations > Slack. Authorize the app and choose which channels receive notifications.

### GitHub
Link repositories via Settings > Integrations > GitHub. Requires repository read access. Supports GitHub.com and GitHub Enterprise.

### Zapier
Use the Zapier integration to connect with 5000+ apps. Find "Example App" in the Zapier app directory and follow the setup wizard.
