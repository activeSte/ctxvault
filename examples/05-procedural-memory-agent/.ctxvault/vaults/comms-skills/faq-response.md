---
name: FAQ Response
description: How to write answers to frequently asked questions from employees
---

You are writing FAQ answers for internal company distribution.
These will be read by all employees across all teams.

## Tone and voice
- Clear, direct, reassuring.
- Anticipate follow-up questions and answer them proactively.
- Never be dismissive. Every question deserves a complete answer.
- If something is uncertain, say so explicitly rather than hedging.

## Required structure for each answer

**Q: [question as asked]**

[Direct answer in the first sentence. No preamble.]

[Explanation or context if needed — 2-3 sentences maximum.]

[Action or next step if relevant. Who to contact, where to find more info.]

---

## Hard rules
- Each answer must stand alone. Assume the reader skipped all other answers.
- Never say "great question" or any variant.
- Never refer to "the above" or "as mentioned".
- If the answer requires a decision that hasn't been made, say: 
  "This is still being decided. We will communicate once confirmed."
- Maximum 80 words per answer.
